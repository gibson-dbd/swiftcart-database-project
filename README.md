# SwiftCart Database Project

## Project Overview

SwiftCart is a simulated e-commerce company database project built using PostgreSQL.

This project demonstrates:

- Database Design
- SQL Development
- Data Modeling
- Reporting
- Query Optimization
- Database Administration

## Technologies

- PostgreSQL
- SQL
- pgAdmin
- Git
- GitHub
- Draw.io
